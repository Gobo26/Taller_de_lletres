# Taller de Lletres

Taller de Lletres és l'espai de poemes, lletres originals i cançons en català de Gobo26: havaneres, boleros, rumbes i valsets per escoltar i llegir.

Web: https://gobo26.github.io/Taller_de_lletres/

## Contingut

- `index.html`: portada amb poemes, lletres i reproductor d'àudio.
- Cançons (mp3): *Agafa fort el timó*, *El vers final*, *La Tona i el gat de mar*, *Paraules sense dir*, *Vela blanca*.
- `manifest.json` i icones per instal·lar el lloc com a aplicació web.

## Encàrrecs i dedicatòries

Si vols una lletra original per una ocasió concreta (aniversari, casament, comiat, homenatge), escriu-me i en parlem.

**Contacte:** [fustaventura@gmail.com](mailto:fustaventura@gmail.com)

## Criteri

Lletres pensades per ser cantades o llegides en veu alta, properes a la tradició mediterrània i amb cura per la sonoritat i el ritme del català.
